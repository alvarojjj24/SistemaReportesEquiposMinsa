# Sistema de Reportes de Equipos - MINSA

Este proyecto es parte de un sistema diseñado para manejar reportes de equipos médicos, utilizando Android como plataforma base.

## Estructura Inicial

La estructura del proyecto incluye los siguientes directorios principales:
- `activities`: Actividades principales de la aplicación, como `MainActivity`.
- `adapters`: Adaptadores para vistas como RecyclerView.
- `models`: Modelos de datos utilizados en la aplicación, por ejemplo, `ReporteEquipo`.
- `database`: Clases para manejo de base de datos utilizando Room.
- `utils`: Utilidades varias para la aplicación.

## Requisitos
- Android Studio Flamingo o superior.
- Gradle 8.
